# RecruitFlow Web

Next.js 15 frontend for the RecruitFlow AI applicant tracking system.

## Local Development

Run the web app from the repository root so workspace dependencies resolve correctly:

```bash
pnpm --filter @recruitflow/web dev
```

The app starts on [http://localhost:3000](http://localhost:3000) and expects the API to be available at `NEXT_PUBLIC_API_URL`.

For full-stack setup, database seeding, and demo credentials, see the root README.
